Wyprowadzenie transmitancji silnika DC

Wejście: U, wyjście: ω (oraz wariant dla kąta θ).

1) Równania modelu

Równanie elektryczne:

\[ U = R i + L \dot{i} + c\,\omega \]

Równanie mechaniczne:

\[ J \dot{\omega} = k i - B\omega - M_o \]

Do wyznaczenia transmitancji przyjmujemy brak momentu obciążenia:

\[ M_o = 0 \]

2) Transformata Laplace’a

Przechodzimy do dziedziny operatorowej (warunki początkowe równe zero):

\[ U(s) = (R + Ls)\,I(s) + c\,\Omega(s) \]
\[ Js\,\Omega(s) = k\,I(s) - B\,\Omega(s) \]

3) Eliminacja prądu

Z równania mechanicznego:

\[ k\,I(s) = (Js + B)\,\Omega(s) \]
\[ I(s) = \frac{(Js + B)}{k}\,\Omega(s) \]

Podstawiamy do równania elektrycznego:

\[ U(s) = (R + Ls)\frac{(Js + B)}{k}\Omega(s) + c\,\Omega(s) \]
\[ U(s) = \left(\frac{(R + Ls)(Js + B)}{k} + c\right)\Omega(s) \]

4) Transmitancja prędkości

Stąd:

\[ \frac{\Omega(s)}{U(s)}= \frac{k}{(Ls + R)(Js + B) + kc} \]

Rozwijamy mianownik:

\[ (Ls+R)(Js+B)+kc = LJ\,s^2 + (LB+RJ)s + (RB+kc) \]
\[ \boxed{ \frac{\Omega(s)}{U(s)}= \frac{k}{LJ\,s^2 + (LB + RJ)s + (RB + kc)} } \]
Układ dla prędkości jest II rzędu i bez astatyzmu (brak całkowania w torze).

5) Transmitancja położenia kątowego

Jeśli wyjściem jest kąt wału θ, to:

\[ \omega = \dot{\theta} \quad\Rightarrow\quad \Theta(s)=\frac{1}{s}\Omega(s) \]

Zatem:

\[ \boxed{ \frac{\Theta(s)}{U(s)}= \frac{1}{s}\frac{\Omega(s)}{U(s)}= \frac{k}{ s\left(LJ\,s^2 + (LB + RJ)s + (RB + kc)\right) } } \]
Układ dla położenia jest III rzędu i ma astatyzm I rzędu (jedno całkowanie: 1/s).
